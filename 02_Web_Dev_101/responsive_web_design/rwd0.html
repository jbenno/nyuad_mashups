Assignments

Assignment until Wednesday Find web pages and "dismantle" their HTML code - delete and add new lines, mashup different pages, etc. Look at Mark Napier's mashup web tools Riot and Shredder and understand what they actually do. Assignment until Monday the following week Look at webdesign.tutsplus.com/series/design-school-for-developers which outlines basic principles of a design process for web sites. Look at some CSS files (e.g. the style sheet of The Well, etc.). Look at geocitiesizer, put in the URL of different websites and try to find out what it does. Scetch a mock web development project for a site that does something you find interesting or funny.

Resize the browser window. Try on different devices.
